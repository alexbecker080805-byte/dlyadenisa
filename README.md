# Screamer Button Web App

Это веб-версия приложения с кнопкой, которая воспроизводит звук.

## Локальный запуск

1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите: `python app.py`
3. Откройте http://127.0.0.1:5000 в браузере.

## Развертывание в интернете

1. Зарегистрируйтесь на [render.com](https://render.com).
2. Создайте новый веб-сервис.
3. Подключите GitHub-репозиторий (загрузите этот код в Git).
4. Укажите команду запуска: `python app.py` (или `gunicorn app:app` для продакшена).
5. Разверните — получите публичный URL.

## Troubleshooting

- Если звук не играет, проверьте настройки браузера (громкость может быть ограничена).
- Убедитесь, что файл `skrimer-golden-freddi-fnaf-1.mp3` в папке `static/`.
